## Packages
framer-motion | Smooth animations for page transitions and card entries
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
- Using Google Fonts: 'Outfit' (Headings) and 'Inter' (Body)
- Authentication is handled via Replit Auth (useAuth hook)
- Doctor images will fallback to UI avatars if not provided
- Distance calculation uses Haversine formula on the client side
- Location services use browser Geolocation API as a fallback for Google Maps
